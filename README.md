# AngularExam-1

Setting up database connection ----------
1. Go to backend/database/mongoose.js
2. Change the string 'changeConnection' inside the mongoose.Connect

Run the Backend API-----------------
1. Go to backend folder
2. npm install
2. nodemon app.js

Run the Frontend----------------
1. Go to frontend folder
2. npm install
3. ng serve
